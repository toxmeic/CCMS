# Campus Complaint Management System (CCMS)

A MERN Stack web application developed to simplify complaint registration and management within a college campus.

## Features

- Student Registration with Email OTP Verification
- Secure Login
- Forgot Password
- Complaint Submission
- View My Complaints
- Edit Profile
- Admin Dashboard
- Update Complaint Status
- Email Notifications on Status Updates
- Search, Filter and Sort Complaints

## Tech Stack

### Frontend
- React
- TypeScript
- CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Other
- Nodemailer
- React Router
- React Icons

## Project Structure

```
CCMS
├── frontend
└── backend
```

## Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm start
```
